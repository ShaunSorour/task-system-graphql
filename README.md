
# Task API

Basic task management server using:

- GraphQL
- PostgreSQL (Docker)
- Apollo Server
- TypeORM
- JWT Auth

<img src="./src/assets/doc.png" alt="Screenshot" width="600" height="600">


## Getting started
1. Install dependencies
```bash
npm install
```
2. Run Project
```bash
npm run start
```
